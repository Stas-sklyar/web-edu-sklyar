# Application for accounting finance

## Capabilities:
	• Adding your daily expenses and income
	• You can see the history, adding expenses and income
	• You can also see a convenient chart of your expenses

## Technologies used:
	• React
	• Redux 
	• TypeScript 
	• Material UI 
	• Jest
